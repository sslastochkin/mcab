# User Guide

!!! info "Installation"
    ```
    pip install mcab
    ```

`mcab` is a library for a/b tests design.  

Its documentation comes in two complementary parts:

* **Tutorial** — the fast and straightforward way to learn the key `mcab`
  concepts hands-on.
* **User Guide** — a problem-oriented reference that focuses on specific tasks
  and approaches in depth.

Reading the tutorial before the user guide is recommended but not required.
The tutorial gives you the general picture, and the guide lets you refine it.

!!! tip
    Use the guide as a reference: jump straight to the section that matches the
    problem you are solving.
